# StartUsingJava

A basic Java JUnit project to check Java and Maven are installed correctly.

Clone or download the project.

Pre-requisites:

* JDK Installed
* MAVEN Installed

In the root folder where the `pom.xml` resides, run the following command:

~~~~~~~~
mvn test
~~~~~~~~
